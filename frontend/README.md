# Cricket Live Frontend

This folder contains a Next.js 14 (App Router) + TypeScript + Tailwind CSS frontend for a cricket live tracker.

Quick start

1. Install dependencies

```bash
cd frontend
npm install
```

2. Run development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
npm start
```

Notes
- Replace the API base URL in `src/services/api.ts` to point to your backend.
- After installing `@types/react` and proper dependencies you can remove local shims added for TypeScript in `src`.
